# Installation
1. Install ruby 2.4.0
2. Run `gem install bundler`
3. Run `bundle update`
4. Run `bundle install`

# DotEnv File
```
DEVELOPMENT_SECRET_KEY_BASE=<development key>
TEST_SECRET_KEY_BASE=<test key>
RAILS_MIN_THREADS=3
RAILS_MAX_THREADS=16
WEB_CONCURRENCY=3
APP_PRELOAD=true
```
